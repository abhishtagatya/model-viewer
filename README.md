# Model Viewer (glTF)

![](docs/preview.gif)

### Quick Build & Run

```bash
cmake . & make & ./model_viewer
```

### Controls

- W, A, S, D + Mouse Movement - Basic Move Camera
- Shift - Accelerate
- 1 - Direct Light Mode
- 2 - Point Light Mode
- 3 - Spot Light Mode
- Arrow Up & Down - Adjust Ambience Value (0.05+-)
